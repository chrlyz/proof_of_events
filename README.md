# Proof Of Events

Mina smart contract and zkPrograms to power an auditable, resilient, and credibly neutral social media platform.

## Clone

```sh
git clone git@github.com:chrlyz/proof_of_events.git
```

## Install

```sh
npm install
```

## Build

```sh
npm run build
```

## Run tests

```sh
npm run test
npm run testw # watch mode
```

## Run coverage

```sh
npm run coverage
```
